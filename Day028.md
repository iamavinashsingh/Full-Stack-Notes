
# Internet & HTML Revision Notes

## How the Internet Works
- **Core Problem**: Getting data between computers not in same location
- **Solution**: Digital signals (1s/0s) over cables
- **Local Networks**: Switch connects devices (like a conference operator)
- **Internet**: Routers connect networks (like postal sorting facilities)
- **IP Addresses**: Unique device identifiers (IPv4: 192.168.1.1, IPv6: 2001:0db8:85a3::)
- **Packets**: Data broken into small pieces for reliable transmission
- **DNS**: Translates domain names (google.com) to IP addresses

## HTML Essentials
- **Structure**: `<html>`, `<head>`, `<body>`
- **Text Elements**: Headings (`h1`-`h6`), Paragraphs (`p`), Breaks (`br`)
- **Media**: Images (`img`), Horizontal rules (`hr`)
- **Lists**: Ordered (`ol`), Unordered (`ul`), Items (`li`)
- **Links**: Anchor tags (`a`) with `href` attribute
- **Forms**: User input collection with various input types

## Key Points to Remember
- Use semantic HTML for better accessibility
- Always include alt text for images
- Validate forms on both client and server side
- Use relative paths for project files, absolute paths for external resources
- HTML creates structure, CSS adds style, JavaScript adds interactivity

## üåê What is CSS?
- CSS = Cascading Style Sheets  
- HTML ‚Üí Content | CSS ‚Üí Appearance  
- Makes websites pretty (colors, fonts, layouts)

---

## ‚ùì Why Need CSS?
- Without CSS ‚Üí plain, boring, hard, messy  
- With CSS ‚Üí clean, reusable, easy to manage

---

## ‚ûï How to Add CSS
1. Inline ‚Üí `<h1 style="color:blue;">...`
2. Internal ‚Üí `<style> h1 {color:blue;} </style>`
3. External (Best) ‚Üí `<link rel="stylesheet" href="styles.css">`

---

## üß© CSS Building Block
```css
selector {
  property: value;
}
```
- Selector = who | Property = what | Value = how

---

## üéØ Selectors
1. Element ‚Üí `p {color:gray;}`
2. Class ‚Üí `.warning {color:red;}`
3. ID ‚Üí `#header {background:blue;}`
4. Group ‚Üí `h1,h2,h3 {font-family:Arial;}`

---

## üé® Colors
- Names ‚Üí `red, lightblue`  
- Hex ‚Üí `#FF0000, #0000FF`  
- RGB ‚Üí `rgb(255,0,0)` / `rgba(255,0,0,0.5)`  
- HSL ‚Üí `hsl(0,100%,50%)` / `hsla(...,0.5)`

## ‚úçÔ∏è Text Styling

## üìê Size Units
- % ‚Üí relative to parent

---

## üì¶ Box Model
- Content ‚Üí text/image  
- Padding ‚Üí inside space  
- Border ‚Üí edge  
- Margin ‚Üí outside space  

Control size: `width`, `height`, `max-width`  

---

## üß† Spacing & Borders
- Padding: `10px` / `10px 20px` / `10px 20px 30px 40px`  
- Margin ‚Üí same shorthand  
- Border ‚Üí `2px solid black;`  
- Border Radius ‚Üí `10px` or `10px 5px 10px 5px`  
- Box-Sizing ‚Üí `*{box-sizing:border-box;}`

---

## üß± Display Types
- Block ‚Üí new line, full width (`div,p,h1`)  
- Inline ‚Üí same line, auto width (`span,a`)  
- Inline-block ‚Üí inline + width/height  
- Hide ‚Üí `display:none;`

---

## üîÑ Block vs Inline
- Block: new line ‚úÖ full width ‚úÖ margin+padding all sides ‚úÖ  
- Inline: same line ‚ùå width/height ignored ‚ùå top/bottom margin ignored ‚ùå

---

## üß¨ CSS Cascade (Priority)
1. `!important` üö®
2. Inline style
3. ID selector
4. Class selector
5. Element selector
6. Source order (last wins)

---

## üìç Position
1. static ‚Üí default, normal flow
2. relative ‚Üí nudges, keeps place
3. absolute ‚Üí free, inside positioned parent
4. fixed ‚Üí stuck to viewport
5. sticky ‚Üí relative + fixed after scroll

---

## üéØ Flexbox (1D Layout)
- Solves vertical centering, equal height, spacing

### Flex Container
- `display:flex;`
- `flex-direction: row/column`
- `justify-content: start/end/center/space-between/around/evenly`
- `align-items: start/end/center/stretch`
- `flex-wrap: wrap/nowrap`
- `gap: 20px`

### Flex Items
- `flex-grow: 1` ‚Üí expand
- `flex-shrink: 0` ‚Üí don‚Äôt shrink
- `flex-basis: 200px` ‚Üí default size
- `flex: grow shrink basis`
- `align-self: flex-start/center/end`

---

## üóùÔ∏è Key Takeaways
- Everything = box, use `box-sizing:border-box;`  
- Padding = inside, Margin = outside  
- Block stacks, Inline flows  
- Cascade priority: `!important > inline > ID > class > element > source order`  
- Position: static, relative, absolute, fixed, sticky  
- Flexbox = simple, responsive 1D layouts
